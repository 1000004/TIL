
# I/O

* 영속적 저장소 (vs휴발성 저장소 저장 공간이 없었다)

    * DataBase - file 차이점 데이터 공유 가능
    * 일회성이 아닌 이전 데이터를 이어받아 연속적으로 이용
    * file을 컨트롤이 가능해지면 프로그램에서 할수 있는게 많아진다
